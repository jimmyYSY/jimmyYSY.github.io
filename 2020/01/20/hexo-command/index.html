hexo创建新页面

  • 在本地hexo目录下打开cms窗口并输入 hexo new (布局) (名字) 即可创建

  • 常用的hexo布局方式有 post、page、draft，分别代表的意义为 文章，页面以及草稿具体对应目录如下：

    布局 路径
    post source/_posts
    page source
    draft source/_drafts
