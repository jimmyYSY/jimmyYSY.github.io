内置的其他方法

  • node-cron 内置共有三个参数
参数 内容 类型
表达式 （* * * * *） String
功能 function(){} Function
options 配置项 Object
  • 配置项为

** scheduled: 设置任务是否以启用，默认为true

** timezone: 配置任务执行时间所在时区

  • start

启动方法

  • stop

    停止方法，除非手动重启任务，否则任务不会继续执行

  • destroy

    任务销毁方法，任务将被停止，并完全销毁，可以通过重新启动来继续任务

  • validate

    验证表达式方法，用来验证表达式是否可用
